What is App Delivery?

The Beeylo App Delivery feature allows your customers to receive order updates directly in the Beeylo mobile app instead of via email. This provides a better customer experience with real-time push notifications and centralized order tracking.

In-App Notifications

Orders appear in the Beeylo app with real-time updates

Push Notifications

Instant alerts for order confirmations and shipping updates

Email Suppression

Optionally disable emails when customers opt for app delivery

Customer Choice

Simple checkbox at cart page for customer opt-in

How It Works
  1. Customer Visits Cart The Beeylo script automatically injects a checkbox: "Receive order updates in Beeylo app instead of email"
  2. Customer Opts In If checked, the preference is saved as a cart attribute: Receive_in_Beeylo_App = Yes
  3. Customer Completes Checkout The cart attribute is passed to the order as a note attribute
  4. Order Created Shopify fires the orders/create webhook to Beeylo
  5. Notification Routing Beeylo detects the preference and sends notification to the app (or email if not opted in)

Important: The script is automatically enabled. No manual theme edits required! However, you may want to test the checkout flow to ensure the checkbox appears correctly on your theme.

Shopify Flow Integration (Plus Only)

Note: Shopify Flow is only available for Shopify Plus merchants. If you don't have Plus, skip to the "Non-Plus Alternatives" section below.

Use Shopify Flow to automatically suppress email notifications when customers opt for app delivery. This ensures they don't receive duplicate notifications.

Setup Workflow: Suppress Order Confirmation Email
  1. Open Shopify Flow Go to: Shopify Admin → Apps → Flow → Create workflow
  2. Set Trigger Trigger: Order created
  3. Add Condition IF Order → Note attributes → Contains "Receive_in_Beeylo_App" AND Contains "Yes"
  4. Add Action Action: Add tag to order
    Tag: beeylo-app-delivery
  5. Add Note (Optional) Action: Add note to order
    Note: "Customer opted for Beeylo app delivery - email notifications suppressed"
  6. Save Workflow Name: "Beeylo App Delivery - Suppress Email"
    Status: Active

Non-Plus Alternatives

If you don't have Shopify Plus, here are alternative options:

Option 1: Disable All Emails

Settings → Notifications → Uncheck "Automatically send order confirmation"

Warning: This disables emails for ALL customers

Option 2: Third-Party Apps

Use apps like Klaviyo, Omnisend, or Brevo with conditional logic to handle email suppression

Option 3: Accept Duplicate Notifications

Let Beeylo send app notifications while Shopify still sends emails (customers get both)

Troubleshooting
Common Issues
  • Script not appearing on cart page

    Check browser console for errors. Verify script URL is accessible at /public/beeylo-cart-integration.js

  • Cart attribute not being captured

    Inspect cart form. Use fetch('/cart.js') in console to verify attributes are set

  • Notifications not being delivered

    Check if customer has beeylo_user_id linked in the database. Verify FCM tokens are valid
